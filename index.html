1.1 Задача линейного алгоритма

Условие: Вкладчик положил в банк 50 000 рублей под 6% годовых. Какая сумма будет на счету через 3 года (простой процент)?

Технологии: Java
class InvalidInputException extends Exception { public InvalidInputException(String message) { super(message); } } public class BankDeposit { public static void main(String[] args) { double P = 50000; double r = 0.06; int t = 3; double S = P * (1 + r * t); System.out.println("Итоговая сумма: " + S + " рублей"); } }

1.2 Задача разветвляющегося алгоритма

Условие: Определите, является ли символ заглавной или строчной буквой.

Технологии: Java
import java.util.Scanner; public class Main { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); char ch = scanner.next().charAt(0); if (ch >= 'A' && ch <= 'Z') { System.out.println("Заглавная буква"); } else if (ch >= 'a' && ch <= 'z') { System.out.println("Строчная буква"); } else { System.out.println("Не буква"); } } }

1.4 Задача на проектирование классов

Условие: Класс "Логин-система" Разработайте класс LoginSystem с приватным полем password и публичными методами для регистрации и входа.

Технологии: Java
import java.util.Scanner; class User { private String username; private String passwordHash; public User(String username, String passwordHash) { this.username = username; this.passwordHash = passwordHash; } public String getUsername() { return username; } public String getPasswordHash() { return passwordHash; } } class AuthService { private User registeredUser; public void register(String username, String password) { String hash = Integer.toString(password.hashCode()); this.registeredUser = new User(username, hash); } public boolean login(String username, String password) { if (registeredUser == null) return false; String hash = Integer.toString(password.hashCode()); return registeredUser.getUsername().equals(username) && registeredUser.getPasswordHash().equals(hash); } } public class LoginSystem { public static void main(String[] args) { AuthService auth = new AuthService(); Scanner scanner = new Scanner(System.in); System.out.println("Регистрация: логин - "); String login = scanner.nextLine(); System.out.println("Пароль - "); String password = scanner.next(); auth.register(login, password); System.out.println("Вход"); System.out.println("Логин - "); String ACK_login = scanner.next(); System.out.println("Пароль - "); String ACK_password = scanner.next(); boolean result = auth.login(ACK_login, ACK_password); System.out.println("Вход успешен: " + result); } }

1.6 Ассемблер 2 задача

Условие: Реализовать процедуру Psub(x,a), которая реализует оператор x := x - a. Передача параметров в стеке, стандартные соглашения о связях. Критерии оценки: запись результата в стек, а не в сегмент данных (неумение работать с параметром-переменной) — минус; каждая другая ошибка — минус 1/3.

Технологии: Assembly (NASM 2.16)
.386 .model flat, stdcall option casemap:none include masm32rt.inc .data x DWORD 20 a DWORD 7 .code Psub PROC uses eax, px:PTR DWORD, pa:PTR DWORD mov eax, [px] mov eax, [eax] mov ecx, [pa] sub eax, [ecx] mov edx, [px] mov [edx], eax ret Psub ENDP start: push OFFSET a push OFFSET x call Psub invoke crt_printf, chr$("x - a = %d\n"), x invoke ExitProcess, 0 end start
